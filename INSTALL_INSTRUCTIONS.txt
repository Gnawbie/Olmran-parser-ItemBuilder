GAMING LOG PARSER - INSTALLATION INSTRUCTIONS
================================================

WHAT'S IN THIS PACKAGE
-----------------------
  OlmranItemBuilder.exe   - the entire program, self-contained

That's it - as of v5.0 the program ships as a single standalone .exe with
Python, tkinter, openpyxl, Pillow, and the bundled community equipment list
all packed inside it. There is nothing to install and nothing else to keep
alongside it.


FIRST-TIME INSTALL
-------------------
1. Put OlmranItemBuilder.exe anywhere on your computer (Desktop, Documents,
   a USB stick, wherever you like - it does not need Administrator rights
   or a special location).
2. Double-click it.
3. Windows SmartScreen will likely show a blue "Windows protected your PC"
   warning the first time, because the .exe isn't code-signed. Click
   "More info", then "Run anyway". This is a one-time warning per file.
4. The Gaming Log Parser window opens. You're done - there's no separate
   install step and nothing to run beforehand.

The first launch takes a few seconds longer than normal while the .exe
unpacks itself into a temporary folder - this happens every time you run
it (it's how a "onefile" packaged program works), not just the first time.


EVERYDAY USE
-------------
Just double-click OlmranItemBuilder.exe. That's the whole program.


WHERE THE PROGRAM STORES YOUR SETTINGS
------------------------------------------
The program saves your last-used folders and Build tab defaults (armor,
weapon, and dual-wield preferences) to a small config file at:

  %USERPROFILE%\.gaming_log_parser_config.json

(that's your Windows user folder, e.g. C:\Users\YourName\.gaming_log_parser_config.json)

This file is created automatically the first time you close the program and
updated whenever you use "Set as Default" in the Build tab. Nothing else is
written anywhere on your system.

To fully uninstall: delete OlmranItemBuilder.exe, and optionally delete the
config file above if you don't plan to reinstall later.


TROUBLESHOOTING
-----------------
"Windows protected your PC" / SmartScreen warning:
  - Expected for an unsigned .exe. Click "More info", then "Run anyway".

Program won't start / nothing happens when double-clicked:
  - Give it a few seconds - a onefile .exe unpacks itself before its window
    appears, and that can take longer on a slow or heavily-loaded drive.
  - Try right-clicking it and "Run as Administrator" as a test (the program
    itself doesn't need admin rights, but this can rule out a permissions
    issue with the folder you put it in).

Community equipment list not found / "Loaded 0 items":
  - The community list is bundled inside the .exe itself in v5.0, so this
    should not happen. If it does, re-download the .exe - it may not have
    downloaded completely.
