# Security Policy

## Supported Versions

Only the latest release is supported. Please update to the newest version
before reporting an issue.

## Reporting a Vulnerability

This is a local desktop tool (parses log files and reads/writes local Excel
files) with no network access or remote services, so its security surface is
small. That said, if you find a security issue - for example something in the
file-parsing or Excel-loading code that could be exploited via a malicious
log/xlsx file - please report it privately rather than in a public issue:

- Use GitHub's [private vulnerability reporting](https://github.com/Gnawbie/Olmran-parser-ItemBuilder/security/advisories/new)
  ("Security" tab → "Report a vulnerability")

Please do not open a public issue for security reports until a fix is
available.
