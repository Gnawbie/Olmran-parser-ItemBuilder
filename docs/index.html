Olmran Item Builder

Parse Olmran/MUD-style game logs, build a master loot database, and search it for the best equipment for your character.

Always the latest release · Windows, portable, no install required beyond Python
Download GamingLogParser.zip Browse files on GitHub All releases

What it does

  • Parses CHAT and ACTION logs into Chat, Combat, and Loot data
  • Builds and maintains a Master Loot Database in Excel
  • Item Builder: search that database for the best gear under armor, weapon, level, and realm constraints, with alternate build options
  • Search Logs: find every time an item actually dropped across a whole batch of logs, with a snapshot of what led up to it

Getting started

  1. Download the zip above and extract it anywhere on your computer
  2. Double-click GamingLogParser_Portable_DEBUG_AutoInstall.bat the first time (installs required packages, then launches)
  3. After that, use GamingLogParser_Portable.bat to launch normally

Full setup details (including what gets installed) are in INSTALL_INSTRUCTIONS.txt, and everything about using the program is in the README.
